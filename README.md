# Project-Nova-Embedded-Vehicle-Control-System

This is a C++ based embedded systems project focused on building and programming a robot car using the OSOYOO Robot Car kit with Arduino. The project follows a step-by-step development process, including hardware assembly, sensor integration, and progressive software implementation. To extend the system beyond the base kit, the project is being enhanced using components from the ELEGOO UNO R3 Super Starter Kit, allowing for expanded experimentation with additional sensors, modules, and control logic. Rather than functioning as a static kit build, this project is designed as an evolving embedded platform. It serves as a foundation for exploring real-time hardware interaction, sensor-based decision making, and autonomous behavior development. Future improvements will focus on expanding functionality through additional sensor integration, refined movement control, and iterative enhancements to autonomous performance. The project emphasizes continuous engineering development through testing, debugging, and incremental system upgrades.
